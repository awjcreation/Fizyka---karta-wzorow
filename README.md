# Fizyka — karta wzorów

Opracowanie edukacyjne inspirowane zakresem kart wzorów do egzaminu maturalnego publikowanych przez Centralną Komisję Egzaminacyjną (CKE).

Układ, forma oraz sposób prezentacji treści zostały opracowane samodzielnie.  
Materiał nie stanowi oficjalnego dokumentu CKE i ma charakter pomocniczy.

---

## 📱 Funkcjonalności

- przegląd działów fizyki w formie mobilnej
- szybkie wyszukiwanie treści
- tryb offline (PWA)
- możliwość instalacji na urządzeniu
